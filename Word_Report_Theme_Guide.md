## üîß How to Apply in Microsoft Word

### Step 1: Create Custom Color Palette
1. **Design Tab** ‚Üí **Colors** ‚Üí **Customize Colors**
2. Set the following:
   - **Text/Background - Dark 1**: #2d3748
   - **Text/Background - Light 1**: #ffffff
   - **Accent 1**: #1f4e79 (Primary Blue)
   - **Accent 2**: #2e75b6 (Secondary Blue)
   - **Accent 3**: #4f81bd (Accent Blue)
   - **Accent 4**: #28a745 (Success Green)
   - **Accent 5**: #fd7e14 (Warning Orange)
   - **Accent 6**: #dc3545 (Error Red)
3. **Save as**: "Computer Garage Theme"

### Step 2: Create Custom Font Set
1. **Design Tab** ‚Üí **Fonts** ‚Üí **Customize Fonts**
2. **Heading Font**: Calibri
3. **Body Font**: Calibri
4. **Save as**: "Computer Garage Fonts"

### Step 3: Apply Heading Styles
1. **Home Tab** ‚Üí **Styles** ‚Üí **Modify Styles**
2. For each heading level, set:
   - Font family and size (as specified above)
   - Color (using your custom palette)
   - Spacing and formatting

### Step 4: Create Table Style
1. **Table Tools** ‚Üí **Design** ‚Üí **New Table Style**
2. Apply the colors and formatting specified above
3. **Save as**: "Computer Garage Table"

## üí° Pro Tips

1. **Consistency**: Use the same colors throughout the document
2. **Hierarchy**: Maintain clear visual hierarchy with font sizes
3. **White Space**: Don't overcrowd - use adequate spacing
4. **Alignment**: Keep consistent alignment (usually left for body text)
5. **Print Test**: Always test how it looks when printed
6. **Accessibility**: Ensure sufficient contrast for readability
7. **Professional**: Avoid too many colors - stick to 2-3 main colors
